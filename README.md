## 배포 가이드 (GitHub Pages)

GitHub Pages를 사용하면 무료로 정적 웹사이트를 쉽게 배포할 수 있습니다.

1.  **GitHub 저장소 생성**:
    *   GitHub에 로그인하여 새로운 저장소를 생성합니다. (저장소 이름은 `[사용자명].github.io` 로 설정하면 좋습니다.)

2.  **로컬 프로젝트와 연결**:
    ```bash
    git init
    git remote add origin https://github.com/[사용자명]/[저장소명].git
    git branch -M main
    ```

3.  **파일 추가 및 커밋**:
    ```bash
    git add .
    git commit -m "첫 커밋: 포트폴리오 기본 구조 생성"
    ```

4.  **GitHub에 푸시**:
    ```bash
    git push -u origin main
    ```

5.  **GitHub Pages 활성화**:
    *   GitHub 저장소의 **Settings** > **Pages** 탭으로 이동합니다.
    *   **Source** 섹션에서 브랜치를 `main`으로 선택하고 `/root` 폴더를 선택한 후 **Save** 버튼을 클릭합니다.
    *   잠시 후, 페이지 상단에 웹사이트 주소가 나타납니다. (예: `https://[사용자명].github.io/[저장소명]`)

이제 이 가이드를 따라 자신만의 포트폴리오를 세상에 공개해보세요!
